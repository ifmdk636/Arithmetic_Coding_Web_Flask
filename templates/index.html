Simulasi interaktif algoritma kompresi berbasis rentang probabilitas

Cara Kerja
①
Hitung Frekuensi
Setiap simbol dihitung kemunculannya untuk membentuk distribusi probabilitas.
②
Rentang Kumulatif
Interval [0,1) dibagi sesuai probabilitas kumulatif masing-masing simbol.
③
Penyempitan Range
Setiap simbol mempersempit interval sebelumnya ke sub-range yang sesuai.
④
Output Bit
Bit dikeluarkan saat MSB stabil, dengan E3 scaling untuk mencegah underflow.
Input Teks
0 / 200

⚠
